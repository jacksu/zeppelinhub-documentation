ZeppelinHub Viewer

ZeppelinHub Viewer is a community site for sharing Zeppelin notebooks. As long as the notebook is publicly located, you can directly look around it without browsing Zeppelin Homescreen. Using Viewer, you can also explore some dramatic examples.


How can you use Viewer ?

You can easily find a text input box in ZeppelinHub and ZeppelinHub Viewer main web page. If you have any public notebook for your own ( or any other public notebook ), copy the note.json location and paste it to the input box and click VIEW button.

For example, Zeppelin provides a Zeppelin Tutorial in Zeppelin github repository. Here is a link.

https://github.com/apache/incubator-zeppelin/blob/master/notebook/2A94M5J1Z/note.json


Or just explore

We provide some collections in the ZeppelinHub Viewer you may be attracted. Just explore interesting example notebooks.
